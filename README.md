## 概要

shellでパスワードマネージャーを作成した。
ここでは、ユーザー名、サービス名、パスワード名を作成すると暗号化されたpasswords.txt.gpgにデータが入る
データの暗号化、復号化にはファイルを守る暗号化キーとしてパスフレーズを自分で作成、入力する必要があります。
作成されたファイルは暗号化された状態を維持します。

## 実装方針

### Add Password(add_password)
ここでそれぞれの入力を行います。
この時に既存の暗号化ファイルがある場合は一時復号させてファイルに展開します。
"暗号化ファイルを復号しています..."

### Get Password(get_password)
ここではサービス名をもとにパスワード、ユーザー名を取得します。
この時にファイルにサービスがなければエラー"そのサービスは登録されていません。"
登録に成功していれば、結果を表示します。

## 作業内容

- 最初にパスワードなしの追加する関数と取り出す関数を作成
- その後、暗号化するために暗号化ファイルの復号ロジックの実装
- 新しいエントリを一時ファイルに追記して、暗号化してから保存
- 一時ファイルはcleanupで安全に削除
